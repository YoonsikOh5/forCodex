# 카르디아 온라인 듀얼

`Cardia_rules_a4_EN.pdf`의 핵심 진행 규칙을 참고해 만든 한국어 2인 온라인 웹 프로토타입입니다.

## 실행

```powershell
npm start
```

브라우저에서 `http://localhost:3000`을 열고 방을 만든 뒤, 초대 링크를 상대에게 공유합니다.

## 현재 구현

- 방 코드 기반 2인 접속
- 각자 손패에서 카드 1장 동시 선택
- 공개 후 영향력 비교
- 높은 카드에 인장 배치
- 낮은 카드의 능력 발동
- 지속 능력, 영향력 수정자, 버림 더미, 다음 카드 보너스
- 인장 5개 이상이면 승리

PDF에는 실제 카드 전문 전체가 포함되어 있지 않아, 확인 가능한 룰과 카드명/예시를 바탕으로 웹 플레이가 되도록 한국어 프로토타입 덱을 구성했습니다.
